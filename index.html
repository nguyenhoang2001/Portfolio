Multi-Agent AI Assistant

Fully offline-capable multi-agent AI assistant using a hierarchical CodeAgent architecture (smolagents/HuggingFace). A Manager Agent orchestrates a Web Search Agent (DuckDuckGo) and a RAG-based Retriever Agent, delegating queries intelligently across both. LLM inference runs locally via Ollama. RAG pipeline uses HuggingFace tokenizer-aware chunking, thenlper/gte-small embeddings, and a persisted FAISS vector index. Chat history stored in PostgreSQL via async SQLAlchemy; responses stream via SSE. Fully Dockerized with docker-compose.
